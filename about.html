About

Two readings of the same question.

llm-arena compares how locally-runnable LLMs perform, two ways: a deterministic benchmark whose every verdict is a pure-function check, and a visual arena where the same open-ended prompt is rendered by different models.

Method

No judge, only checks

Benchmark scoring is intentionally boring: each task is graded by a pure function over the model's output — no LLM-as-a-judge, nothing you can't re-run yourself and get the same number. The arena is the opposite register, qualitative and visual, but the prompt is held fixed so the only variable is the model. Methodology detail lands with each pillar.
