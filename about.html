How I measure local LLMs.

llm-arena measures how locally-runnable LLMs perform with a deterministic benchmark: every verdict is a pure-function check over the model's output, so you can re-run it and get the same number.

Method

No judge, only checks

Benchmark scoring is intentionally boring: each task is graded by a pure function over the model's output. No LLM-as-a-judge, nothing you can't re-run yourself and get the same number. Every model runs the same battery, and a gold reference must pass every check before any model is scored against it.
